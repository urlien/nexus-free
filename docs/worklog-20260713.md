## 🔧 技术要点

### LLM 调用架构
```
用户请求 → search路由(多源搜索) → 原始文本
                                       ↓
                          analyzer.py(AI管道)
                                       ↓
              extract_character_info() → CharacterInfo
              synthesize_description() → PLists格式
              generate_dialogue()     → 对话示例
                                       ↓
                          CharacterCard(SillyTavern V2)
```

### API 配置
- 环境变量优先：MIMO_API_BASE, MIMO_API_KEY, MIMO_MODEL
- 默认使用 MiMo Lite 套餐的 mimo-flash
