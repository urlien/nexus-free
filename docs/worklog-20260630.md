## 🔍 kcbuy.cn 技术分析

### 网站信息
- **地址**: https://kcbuy.cn
- **技术栈**: React SPA + Python/Node后端 + Nginx
- **服务器**: nginx/1.18.0 (Ubuntu)
- **AI引擎**: DeepSeek V4 Flash
- **版本**: API v0.2.0

### 认证系统
- JWT Token认证
- Bearer Token格式
- Token格式: 标准JWT格式（Header.Payload.Signature）
- 用户ID在sub字段（如: "92"）

### 合成描述的详细字段要求
来自 `synthesize_description` 工具的prompt：
- `description_synth`: 1200-2000字PLists格式
- `personality_profile`: 400-800字
- `scenario_context`: 200-350字
- `appearance`: 100-300字
- `speech_pattern`: 80-200字
- `skills_abilities`: 150-400字
- `relations`: 至少3个关系
- `likes`: 5+个
- `dislikes`: 5+个
- `habits_quirks`: 4+个
- `goal`: 100-200字（含恐惧）
- `alternate_greetings`: 5条100-200字

### 角色卡导出的详细字段要求
来自 `export_character_card` 工具的prompt：
- `name`: 角色名
- `identity`: 身份
- `aliases`: 别名列表
- `backstory_summary`: 200+字
- `appearance`: 100+字（发色/瞳色/身高/特征/服装）
- `personality_profile`: 400+字（含多维分析+行为例证）
- `speech_pattern`: 语速/口癖/称呼
- `skills_abilities`: 能力+限制
- `relations`: 至少3个关键关系的详细描述
- `likes`: 5+
- `dislikes`: 5+
- `habits_quirks`: 4+
- `goal`: 短期+长期+恐惧
- `tags`: 标签
- `source_works`: 出处

⚠️ 关键字段说明：
- `scenario_context` 和 `first_mes` 是最关键的字段，必须调用 `generate_dialogue` 工具生成后填入，禁止自己写
- `scenario_context`: 200-350字具体场景（时间/地点/氛围/冲突），禁止写占位文字如"场景设定"
- `first_mes`: 250-400字完整开场（环境+角色动作+对话+回应入口），禁止写"我是XX很高兴见到你"
- `mes_example`: 4-6组对话回合，用 generate_dialogue 生成
- `alternate_greetings`: 5条不同情境开场（100-200字每条）
- 多角色卡要求：scenario_context 描述各角色在同一场景中的位置/状态/互动，first_mes 写共同场景中的互动而非各自独立介绍

## 💡 技术要点

### LLM调用模板
```python
import httpx

async def call_llm(prompt: str, system: str = ""):
    """调用DeepSeek/MiMo API"""
    async with httpx.AsyncClient() as client:
        resp = await client.post(
            "https://api.deepseek.com/v1/chat/completions",
            headers={"Authorization": "Bearer YOUR_API_KEY"},
            json={
                "model": "deepseek-chat",
                "messages": [
                    {"role": "system", "content": system},
                    {"role": "user", "content": prompt}
                ]
            }
        )
        return resp.json()["choices"][0]["message"]["content"]
```

### Fandom API调用
```python
# Fandom wiki API
url = f"https://{wiki_name}.fandom.com/api.php"
```
